## ğŸš€ Getting Started

### Prerequisites

- Node.js 18+ installed
- Firebase project with AI services enabled
- Google AI API access
- Follow the [Firebase AI Logic Getting Started Guide](https://firebase.google.com/docs/ai-logic/get-started?api=dev) to set up Firebase AI SDK in your project

### Installation

1. Clone the repository:
```bash
git clone <your-repo-url>
cd roastbot
```

2. Install dependencies:
```bash
npm install
```

3. Create a `.env` file in the root directory with your Firebase credentials:
```env
NEXT_PUBLIC_FIREBASE_API_KEY=your_api_key
NEXT_PUBLIC_FIREBASE_AUTH_DOMAIN=your_auth_domain
NEXT_PUBLIC_FIREBASE_PROJECT_ID=your_project_id
NEXT_PUBLIC_FIREBASE_STORAGE_BUCKET=your_storage_bucket
NEXT_PUBLIC_FIREBASE_MESSAGING_SENDER_ID=your_messaging_sender_id
NEXT_PUBLIC_FIREBASE_APP_ID=your_app_id
```

4. Run the development server:
```bash
npm run dev
```

5. Open [http://localhost:3000](http://localhost:3000) in your browser
